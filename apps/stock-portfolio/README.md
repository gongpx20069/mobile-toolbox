# 美股持仓看板

适合手机横屏常亮展示的纯前端美股持仓看板。行情默认使用腾讯接口，不依赖 Yahoo。

## 功能

- 添加 / 更新持仓：股票代码、数量、成本价
- 本地浏览器保存持仓，不上传券商账号或密码
- 展示总市值、总成本、持仓盈亏、日内变动
- 每 60 秒自动刷新腾讯美股行情
- 支持添加到手机主屏幕作为 PWA 使用
- 支持浏览器 Screen Wake Lock API 时可开启网页常亮

## 本地访问

从仓库根目录启动静态服务：

```powershell
python -m http.server 5173
```

然后打开：

```text
http://localhost:5173/apps/stock-portfolio/
```
